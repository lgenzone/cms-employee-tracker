  ![License badge](https://img.shields.io/badge/license-MIT-green})

  ## Description
  A command-line program that handles a company's employee database using Node.js, Inquirer, and MySQL.

  ## Installation 
  
* Clone the repo
* Install Node.js
* Install the dependencies using the following command: npm install

  ## Usage 

  How to use this application:

* In the terminal, execute npm start or node server.js to launch the program.
* To navigate selections, use the arrow keys, and press 'enter' to select
* Follow the prompts, and press 'enter' when each is completed
* To exit the application, choose 'exit' and press enter.
